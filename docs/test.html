Student Card reader Project

Our goal was to create a simple badge-in system using student cards. We built a device with a Raspberry Pi, RFID reader, and LCD screen. The Raspberry Pi sends card data to an API for name retrieval and exam eligibility check. A USB camera captures a photo for facial recognition, and the system logs check-in dates to a central database via the API. Successful check-ins trigger automated email confirmations to students. We also developed a web interface for teachers to access student info and exam attendance status efficiently.
